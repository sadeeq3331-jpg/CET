# 🇬🇧 English Master – CET Test Prep

> Full mock exams, vocabulary trainers, and interactive tools for Chinese English proficiency tests (CET‑4, CET‑6, 考研英语). Runs entirely in the browser — no server required.

---

## 📊 What's Inside

| Module | Description |
| :--- | :--- |
| **CET‑4 Vocabulary** | 4,500+ headwords with phonetics, Chinese meanings, examples |
| **CET‑4 Phrases** | 1,000+ essential collocations |
| **CET‑4 Mock Exam** | Full‑length timed simulation (Writing, Listening, Reading, Translation) |
| **Grammar Guide** | Key grammar points with examples |
| **Daily Challenge** | Random word quiz with streak tracking |
| **Badges & Progress** | Achievement tracking based on study activity |
| **Interactive Assistant** | Floating quiz/typing buddy for vocabulary practice |
