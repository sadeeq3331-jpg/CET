# 📚 HSK & English Test Prep – Premium Learning Platform

A collection of interactive web‑based mock exams, vocabulary trainers, and grammar guides for Chinese HSK (1‑6) and English proficiency tests (CET‑4/6, 考研英语). Designed for self‑study with audio playback, favorites, daily challenges, and progress tracking – all running entirely in the browser.

---

## ✨ Features

- **Full mock exams** – timed simulations matching official formats (HSK 1‑6, CET‑4, CET‑6, 考研英语)
- **Vocabulary master** – 4500+ words per level, with phonetic, meaning, example, search, letter filter, study mode
- **Phrase library** – 1000+ collocations for CET‑4
- **Grammar guides** – key grammar points with examples
- **Daily challenge** – random word quiz with streak tracking
- **Badges & progress** – achievements based on study activity
- **Interactive assistant** – floating quiz/typing buddy for vocabulary practice
- **Premium system** – one‑time payment gate using Google Sheets backend
- **Dark mode** – theme saved per user
- **No server required** – pure HTML/CSS/JavaScript, runs from any static hosting
